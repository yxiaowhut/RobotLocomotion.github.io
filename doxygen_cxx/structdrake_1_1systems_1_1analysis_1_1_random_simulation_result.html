Detailed Description

A snapshot of the generator used to produce the random simulation.

Use, e.g.,

RandomGenerator generator(result.generator_snapshot)
RandomSimulation(make_simulator, output, final_time, &generator)

for a deterministic playback of the sampled simulation. Use the operator<<() operator>>() methods provided for the generator, which implements the Standard Template Library RandomNumberEngine concept, if you wish to serialize the results. Note that the generator "snapshots" are returned as const, because any non-const operations on that object may advance the state of the generator (making it no-longer valuable in reproducing the simulation).
