Detailed Description

template<typename LcmMessage>
class drake::systems::lcm::Serializer< LcmMessage >

Serializer is specific to a single LcmMessage type, and translates between LCM message bytes and drake::systems::Value<LcmMessage> objects.

Template Parameters
LcmMessagemessage type to serialize, e.g., lcmt_drake_signal.
