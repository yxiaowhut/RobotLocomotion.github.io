Detailed Description

This file assumes a 6-degree of freedom Kinova Jaco arm with a three finger gripper.

Joints are numbered sequentially starting from the base with the following joint index descriptions: 0: shoulder roll 1: shoulder fore/aft 2: elbow fore/aft 3: forearm roll 4: wrist yaw 5: wrist roll 6: finger 1 bend/extend 7: finger 2 bend/extend 8: finger 3 bend/extend

Position units are in radians, velocity units are in radians per second
