Detailed Description

template<typename T>
class drake::systems::PidControlledSpringMassSystem< T >

A model of a one-dimensional spring-mass system controlled to achieve a given target position using a PID controller.

See also
SpringMassSystem, PidController.
Template Parameters
TThe vector element type, which must be a valid Eigen scalar.

Instantiated templates for the following kinds of T's are provided:

  • double
  • AutoDiffXd

They are already available to link against in the containing library. No other values for T are currently supported.
