Detailed Description

template<typename T>
class drake::systems::controllers::qp_inverse_dynamics::RobotKinematicState< T >

A wrapper class around KinematicsCache and several useful matrices such as the inertia matrix, etc.

This class serves mainly as a explicit cache to avoid repeated computation. This class can be replaced when System's cache is ready.
