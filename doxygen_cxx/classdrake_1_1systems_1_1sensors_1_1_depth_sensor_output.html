◆ GetTooCloseDistance()

double GetTooCloseDistance ( )
static

Returns the depth value when the min sensing range is violated because the object being sensed is too close.

Note that this differs from ROS, which uses negative infinity in this scenario. Drake uses zero because it results in less devastating bugs when users fail to check for the lower limit being hit and using negative infinity does not prevent users from writing bad code. This value can be used in conjunction with IsTooClose().
