Eigen::Matrix<typename DerivedQ::Scalar, Eigen::Dynamic, 1> SpringTorque ( const Eigen::MatrixBase< DerivedQ > &  q) const
inline

Compute the spring torque for a simple singleaxis joint.

Since this is a one DOF joint, the input and output vectors will be length 1.

Torque is computed to be included in the dynamics bias terms, and thus appears here with a "positive" gain.
