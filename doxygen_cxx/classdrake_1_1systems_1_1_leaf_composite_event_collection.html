Detailed Description

template<typename T>
class drake::systems::LeafCompositeEventCollection< T >

A CompositeEventCollection for a LeafSystem.

i.e. 

  PublishEvent<T>: {event1i, ...}
  DiscreteUpdateEvent<T>: {event2i, ...}
  UnrestrictedUpdateEvent<T>: {event3i, ...}
