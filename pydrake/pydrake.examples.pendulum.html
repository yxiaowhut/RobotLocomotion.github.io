class pydrake.examples.pendulum.PendulumPlant¶

Bases: pydrake.systems.framework.LeafSystem_[float]

A model of a simple pendulum

\[ml^2 \ddot\theta + b\dot\theta + mgl\sin\theta = \tau\]

@system{PendulumPlant, @input_port{tau}, @output_port{state} @output_port{geometry pose} }

Template parameter T:
The vector element type, which must be a valid Eigen scalar.

Instantiated templates for the following kinds of T’s are provided:

  • double
  • AutoDiffXd
  • symbolic::Expression
