class pydrake.examples.acrobot.AcrobotPlant¶

Bases: pydrake.systems.framework.LeafSystem_[float]

The Acrobot - a canonical underactuated system as described in <a href=”http://underactuated.mit.edu/underactuated.html?chapter=3”>Chapter 3 of Underactuated Robotics</a>.

@system{ AcrobotPlant, @input_port{elbow_torque}, @output_port{acrobot_state} }

Template parameter T:
The vector element type, which must be a valid Eigen scalar.

Instantiated templates for the following kinds of T’s are provided:

  • double
  • drake::AutoDiffXd
  • symbolic::Expression
CalcKineticEnergy(self: pydrake.examples.acrobot.AcrobotPlant, arg0: pydrake.systems.framework.Context_[float]) → float¶
CalcPotentialEnergy(self: pydrake.examples.acrobot.AcrobotPlant, arg0: pydrake.systems.framework.Context_[float]) → float¶
DynamicsBiasTerm(self: pydrake.examples.acrobot.AcrobotPlant, arg0: pydrake.systems.framework.Context_[float]) → numpy.ndarray[float64[2, 1]]¶

Manipulator equation of Acrobot: M(q)q̈ + bias(q,q̇) = B*u.

  • M[2x2] is the mass matrix.
  • bias[2x1] includes the Coriolis term, gravity term and the damping term, i.e. bias[2x1] = C(q,v)*v - τ_g(q) + [b1*q̇₁;b2*q̇₂].
MassMatrix(self: pydrake.examples.acrobot.AcrobotPlant, arg0: pydrake.systems.framework.Context_[float]) → numpy.ndarray[float64[2, 2]]¶
__init__(self: pydrake.examples.acrobot.AcrobotPlant) → None¶

Constructs the plant. The parameters of the system are stored as Parameters in the Context (see acrobot_params.named_vector).
